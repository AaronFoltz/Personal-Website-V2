    • Node HackerNews API

    A simple Node Wrapper API around news.ycombinator.com. The default path is http://hackernews.aaronfoltz.com/get which will return a pure JSON implementation of the front page posts. To get other pages, you can do "http://hackernews.aaronfoltz.com/get?page=#".


    At the moment, you can get the:
  • Rank
  • Title
  • Link URL
  • Points
  • User
  • User Profile URL
  • Posted Time
  • Number of comments
  • Comment URL


  • https://github.com/AaronFoltz/node-hackernews
